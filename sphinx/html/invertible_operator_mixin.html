Bases: object

Mixin class to invert implicit defined operators.

To invert the application of a given implicitly defined operator on a field, this class gives the necessary functionality. Inheriting functionality from this class provides the derived class with the inverse to the given implicitly definied application of the operator on a field. (e.g. .inverse_times vs. .times and .adjoint_times vs. .adjoint_inverse_times)

Parameters:

inverter : Inverter

An instance of an Inverter class. (default: ConjugateGradient)

preconditioner : LinearOperator

Preconditions the minimizaion problem
